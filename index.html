Etiquetas de estilo en HTML

Este es un texto de ejemplo para practicar con las etiquetas de estilo. Vamos a resaltar algunas palabras importantes.

Estilo negritas

La etiqueta bse utliza para poner el texto en negritas por ejemplo: este texto esta en negritas

la etiqueta strong tambien se usa para poner el texto en negritas pero tiene un significado semantico mas fuerte.

Diferencias entre b y strong

Ambas etiquedas hacen el texto se vea en negritas pero los motores de busqueda y los lectores de pantalla pueden utilizar esta informacion para entender mejor el contenido de la pagina

Estilo cursiva

La etiqueta ise utiliza para escribir el texto en cursiva. por ejemplo: este texto esta en cursiva

la etiqueta emtambien se utiliza para poner el texto en cursiva,pero tiene un significado semantico en enfasis

Diferencias entre i y em

Ambas etiqutas hacen que se vea en cursiva pero em altener mayor enfasis semantico el lector de pantalla puede enfatisar la lectura de estas palabras

Estilo subrallado

La etiqueta use utiliza para subrallar el texto. por ejemplo: este texto esta subrayado
